# NodeL2
Node implementation of a L2 server emulator

<h2>How to</h2>

* Install `NodeJS`
* Import `nodel2.dump` backup into `nodel2` SQL database
* Run `npm install -i`
* Run `node LaunchServer`
* Default credentials are username `q` and password `q`
